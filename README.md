# Verma Farms

Front End Code Challenge

## Getting Started

```sh
git clone https://github.com/lemuellloren/coding-challenge-ehrlich.git
```

then:

```sh
cd coding-challenge-ehrlich
npm install
npx sass --watch scss/main.scss css/style.css
```
